Social Media Trend Report

Group 7: Kim Eun Kyeol, Song Ji Woo, Park Jae Sung, Wang Eun Chan

Main Objectives and Features

  1. Present topic-based trend summaries and explain the causes behind the popularity of selected hashtags or keywords.
  2. Include a user sentiment analysis showing the ratio of positive to negative reactions.
  3. Use tables, lists, and figures to display summarized data.
  4. Embed visual elements such as keyword clouds, charts, and representative images or screenshots of posts.
  5. Provide reference links to source data or further readings.

Project Overview

This project analyzes trending hashtags and topics across Google, Twitter, Instagram, and YouTube.
